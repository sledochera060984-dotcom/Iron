# Iron

Осетинский словарь и будущий Android-проект приложения.

## Цель

Сделать приложение-словарь на основе исходного файла словаря.

## План

1. Загрузить исходный файл словаря в `data/raw/`.
2. Преобразовать Word-файл в структурированный формат: JSON, CSV или SQLite.
3. Подготовить локальную базу для приложения.
4. Создать Android-приложение на Kotlin / Jetpack Compose.
5. Добавить поиск по осетинскому и русскому тексту.
6. Добавить карточку слова, избранное и историю поиска.

## Структура

```text
Iron/
├─ app/
├─ data/
│  ├─ raw/
│  └─ processed/
├─ README.md
└─ progress.md
```

## Важно

Исходный Word-файл лучше хранить отдельно как сырой материал. Для приложения удобнее использовать обработанную базу в формате JSON или SQLite.
