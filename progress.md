## Ближайшие задачи

### 1. Загрузка словаря

Нужно добавить исходный файл словаря, например:

```text
data/raw/IronGenDict.doc
```

### 2. Разбор файла

Нужно извлечь из Word-файла записи словаря и привести их к структуре:

```json
{
  "os": "осетинское слово",
  "ru": "русский перевод",
  "grammar": "пометы, если есть",
  "examples": []
}
```

### 3. Формат базы

Первый простой вариант:

```text
data/processed/dictionary.json
```

Более правильный вариант для приложения:

```text
data/processed/dictionary.sqlite
```

### 4. Android-приложение

Планируемый стек:

- Kotlin
- Jetpack Compose
- локальная база JSON или SQLite/Room
- поиск по осетинскому и русскому тексту
- избранное
- история поиска

## Важные решения

- Word-файл не использовать напрямую в приложении.
- Сначала сделать локальный офлайн-словарь.
- Синхронизацию и платные функции добавлять позже, после рабочей первой версии.
