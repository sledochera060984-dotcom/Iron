
## Ближайшие задачи

### 1. Восстановить Excel-файл

В терминале из корня проекта:

```bash
python tools/decode_raw_file.py
```

### 2. Разбор файла

Нужно извлечь из Excel-файла записи словаря и привести их к структуре:

```json
{
  "os": "осетинское слово",
  "ru": "русский перевод",
  "grammar": "пометы, если есть",
  "examples": []
}
```

### 3. Формат базы

Первый простой вариант:

```text
data/processed/dictionary.json
```

Более правильный вариант для приложения:

```text
data/processed/dictionary.sqlite
```

### 4. Android-приложение

Планируемый стек:

- Kotlin
- Jetpack Compose
- локальная база JSON или SQLite/Room
- поиск по осетинскому и русскому тексту
- избранное
- история поиска

## Важные решения
